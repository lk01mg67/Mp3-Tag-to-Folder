# Mp3-Tag-to-Folder

MP3 파일의 메타데이터(Tag)를 분석하여 아티스트와 앨범별로 폴더를 자동 정리해주는 Go 기반 도구입니다.

## 📌 제작 동기

Google Music 서비스 종료 후 백업받은 MP3 파일들이 구조 없이 한 폴더에 모여 있는 문제를 해결하고, Go 언어 학습 및 실전 활용을 위해 제작되었습니다.

## ✨ 주요 기능

- **태그 분석**: `github.com/dhowden/tag` 라이브러리를 활용해 MP3 내의 아티스트, 앨범, 트랙 번호, 제목 추출
- **경로 정규화**: 파일 시스템에서 사용 불가능한 특수문자(`:`, `/`, `?`, `*` 등)를 전각 문자로 자동 치환
- **자동 폴더 구조화**: `[아티스트] / [앨범] / [디스크 번호]` 순으로 계층적 폴더 생성 및 파일 복사
- **안전한 처리**: 기존 파일을 직접 수정하지 않고 새로운 경로로 복사하여 데이터 유실 방지

## 🛠 사용 기술

- **Language**: Go
- **Library**: [dhowden/tag](https://github.com/dhowden/tag)

## 🚀 실행 방법

1. 의존성 설치:

   ```bash
   go mod tidy
   ```

2. 프로그램 실행:

   ```bash
   go run main.go
   ```

   *(현재 경로가 `main.go` 내에 하드코딩되어 있으므로, 실행 전 소스 코드의 경로 설정을 확인하세요.)*
